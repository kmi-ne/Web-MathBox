Let $f: [a, b] \to \mathbb{R}$ be continuous on $[a, b]$ and differentiable on $(a, b)$. If $f(a) = f(b)$, then there exists a point $c \in (a, b)$ such that $f'(c) = 0$.

Since $f$ is continuous on the closed interval $[a, b]$, by the extreme value theorem, $f$ attains its maximum and minimum values on $[a, b]$.

If $f$ is constant on $[a, b]$, then $f'(x) = 0$ for all $x \in (a, b)$, and we can choose any $c \in (a, b)$.

If $f$ is not constant, then either its maximum or minimum must occur at some point $c \in (a, b)$ (since $f(a) = f(b)$).

At such a point $c$, we have $f'(c) = 0$ by Fermat's theorem.

Let $f: [a, b] \to \mathbb{R}$ be continuous on $[a, b]$ and differentiable on $(a, b)$. Then there exists a point $c \in (a, b)$ such that $f'(c) = \frac{f(b) - f(a)}{b - a}$.

Define a new function $g(x) = f(x) - \frac{f(b) - f(a)}{b - a}(x - a) - f(a)$.

Then $g$ is continuous on $[a, b]$ and differentiable on $(a, b)$.

Furthermore, $g(a) = f(a) - f(a) = 0$ and $g(b) = f(b) - \frac{f(b) - f(a)}{b - a}(b - a) - f(a) = 0$.

By , there exists $c \in (a, b)$ such that $g'(c) = 0$.

Since $g'(x) = f'(x) - \frac{f(b) - f(a)}{b - a}$, we have $f'(c) = \frac{f(b) - f(a)}{b - a}$.

If $f: [a, b] \to \mathbb{R}$ is continuous and $f(a) < k < f(b)$ (or $f(a) > k > f(b)$), then there exists a point $c \in (a, b)$ such that $f(c) = k$.

Let $S = \{x \in [a, b] : f(x) \leq k\}$.

Since $f(a) < k$, we know that $a \in S$, so $S$ is non-empty.

Since $f(b) > k$, we know that $b \notin S$, so $S$ is bounded above by $b$.

By the completeness of the real numbers, $S$ has a supremum $c = \sup S$.

We claim that $f(c) = k$.

If $f(c) < k$, then by continuity of $f$, there exists $\delta > 0$ such that $f(x) < k$ for all $x \in (c - \delta, c + \delta) \cap [a, b]$.

But this means that $c + \min(\delta/2, b-c) \in S$, contradicting that $c = \sup S$.

If $f(c) > k$, then by continuity of $f$, there exists $\delta > 0$ such that $f(x) > k$ for all $x \in (c - \delta, c + \delta) \cap [a, b]$.

But this means that $\sup S \leq c - \delta < c$, contradicting that $c = \sup S$.

Therefore, $f(c) = k$.
